# Personal-Site

### Project Summary:
  "Proof of Concepts" project, highlighting the use cases of different HTML elements.
### HTML elements used:
  Heading elements
  Line break elements
  Horizontal rule elements
  Paragraph tags
  Emphasis elements
  Strong tags
  Ordered and unordered lists
  Image elements
  Anchor tags
  Tables
  Forms - labels, inputs, textareas
### Languages and technologies:
  HTML5
